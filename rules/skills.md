# Skills Catalog

The full catalog of installed plugins and skills worth exploring is in:
`~/.claude/bookmarks.md`

## When to check it

- User asks "do I have a skill for X?" or "is there a skill that..."
- You're about to start a task and a specialized skill might improve the output
- User is starting a new type of project (SEO, mobile, system design, etc.)

## How to check it

Read `~/.claude/bookmarks.md`. Installed plugins are under
"Installed Plugins". Skills worth installing for the current job are under "To Explore".
